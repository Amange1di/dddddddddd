Системы предупреждения и предотвращения столкновений:

Mobileye разрабатывает системы, способные обнаруживать препятствия на дороге, включая другие транспортные средства, пешеходов и велосипедистов, и предупреждать водителей о возможных столкновениях.

Системы адаптивного круиз-контроля:

Технологии Mobileye включают в себя системы умного круиз-контроля, которые могут автоматически подстраивать скорость автомобиля в соответствии с движением других транспортных средств и обстановкой на дороге.

Системы распознавания знаков дорожного движения:

Mobileye разрабатывает технологии для распознавания дорожных знаков, включая ограничения скорости, предупреждающие знаки и другие информационные знаки.

Картография для автономных транспортных средств:

Mobileye также работает над созданием детальных карт и картографических данных, необходимых для навигации автономных транспортных средств.

Baidu Apollo – это инициатива компании Baidu, китайского технологического гиганта, по разработке и внедрению технологий для автономного вождения. Проект Apollo нацелен на создание открытой платформы для разработки автономных транспортных средств, предоставляя разработчикам и автопроизводителям доступ к инструментам и ресурсам для создания своих систем.

Основные аспекты Baidu Apollo включают:

Системы искусственного интеллекта:

Baidu Apollo использует технологии искусственного интеллекта, такие как машинное обучение и нейронные сети, для обработки данных и принятия решений в реальном времени.

Сенсоры и оборудование для восприятия окружающей среды:

Транспортные средства, оснащенные Baidu Apollo, обычно оборудованы различными сенсорами, такими как камеры, радары и лидары, для точного восприятия окружающей среды.

Открытая платформа для разработки:

Baidu Apollo предоставляет открытые API и инструменты для разработки, позволяя сторонним разработчикам и автопроизводителям создавать и интегрировать свои решения для автономного вождения.

Тестирование автономных транспортных средств:

Baidu активно тестирует автономные транспортные средства с использованием технологии Apollo на дорогах в различных частях мира.
