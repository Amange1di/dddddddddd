Waymo's DeepMind

Прогнозирование временных рядов

Long Short-Term Memory (LSTM) Networks

+

LSTM - это тип рекуррентных нейронных сетей, который хорошо подходит для работы с временными рядами. Он способен учитывать долгосрочные зависимости в данных. перейте

Gated Recurrent Units (GRU)

+

GRU - это еще один тип рекуррентных сетей, аналогичный LSTM, но более простой по структуре. Он также эффективен в прогнозировании временных рядов.перейте

Convolutional Neural Networks (CNN) for Time Series

+

CNN, изначально созданные для обработки изображений, могут быть адаптированы для анализа временных рядов, выделяя важные временные шаблоны.перейте

Prophet by Facebook

+

Prophet - это инструмент для анализа и прогнозирования временных рядов, разработанный Facebook. Он может использовать как классические методы, так и нейронные сети для прогнозирования данных.перейте

WaveNet

+

WaveNet - это генеративная модель, разработанная DeepMind для работы с аудио-временными рядами. Он может использоваться для генерации звуков и анализа звуковых данных.перейте

PathAI

Компания PathAI применяет нейросети для анализа биоптатов и обнаружения признаков различных заболеваний в тканях.перейте
